# Setup Instructions

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/fredezoe52-prog/OMG--PLATFORME-P.git
   ```

2. Navigate to the project directory:
   ```bash
   cd OMG--PLATFORME-P
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## Configuration
1. Create a `.env` file in the root directory based on the `.env.example`:
   ```bash
   cp .env.example .env
   ```

2. Update the `.env` file with your configuration settings.

3. Start the application:
   ```bash
   npm start
   ```
